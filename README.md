# RCE-2-WF
Прошивка USB Wi-Fi PRO EKF Connect (TYWE3S) под esphome 

Большей своей частью написано при помощи ИИ, я не программист  и не электрик.

Данное устройство  выпускалось с разными чипами:

## Версии модулей

- **v1.0** - ESP8266 (TYWE3S) *неотключаемый блок USB*
- **v1.1** - RTL8710BX
- **v1.1** - BK7231T (CBU) *отключаемый USB*
- **v1.2** - BK7231N (WB2L/WB3S)
- **v2.0** - T34 (WB8S)


Для варианта с BK7231T (CBU) прошивка есть в интернете, мне не повезло, у меня устройство на плате TYWE3S, за энергомониторинг отвечает BL0937 поэтому дальнейшее (без значительных изменений) актуально только для данного чипа

## Прошивка устройства
1. Подключение через USB-UART (в моем случае PL2303), к самой плате нужно подпаяться согласно таблице подключения;
2. *Снять дамп через esptool не вышло, видимо нужно хорошее (или отдельное) питание TYWE3S (но удалось узнать что flash память объемом 2MB), если вам интересно можно этим заняться;*
3. В ESPHome создаем устройство, даем ему имя, выбираем ESP8266, внутри созданного устройства получаем ключи и пароли;
4. Редактируем yaml файл полученный из assets под ваши данные;
5. Компилируем прошивку и прошиваем устройство (через отдельную программу, либо напрямую через браузер в esphome, выбрав - устройство подключено к PC);
6. Проверяем устройство, подключаем нагрузки, редактируем параметры voltage_divider (Напряжение) и current_resistor (Сила тока) под реальные значения (Новое значение = текущее значение * (то что отображается в HA / Реальное, измеренное значение))/
7. Обновляемся по WiFI

## Подключение

|-----------------|-------------|
| 3.3v            | VCC         |
| RX              | TX          |
| TX              | RX          |
| GND             | GND         |
| GND             | GPIO0       |

## Основные функции
*Управление из Home Assistant*
 1. Управление 4 розетками по отдельности или одновременно;
 2. Отслеживание: Силы тока, Напряжения, Мощности, Энергии, Состояния защиты;
 3. Включение/Отключение розеток при подаче питания;
 4. Режим индикации светодиода:
    - Быстро мигает если сработала защита (не зависит от режима индикации);
    - Постоянно горит если включена хотя бы одна розетка;
    - Медленно мигает если теряет сигнал WiFi;
    - Не горит, если все розетки выключены, либо режим индикации отключен.
    
*Управление кнопкой:*
 - Сбрасывает уведомление о сработавшей защите;
 - Включает все розетки если все выключены;
 - Отключает все розетки если включена хотя бы одна;
 - При зажатии более чем на 3 сек изменяет режим при подаче питания (уведомляет световой индикацией).

*Особенности*
 - При одновременном включении всех розеток, розетки включаются последовательно, с задержкой в 200мс;
 - Защита срабатывает при превышении суммарного тока 16А или напряжение выходит за диапазон 190-250V;
 - После отработки защиты, уведомление защиты сбрасывается только при нажатии физической кнопки.
